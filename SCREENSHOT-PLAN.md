## Primary screenshot set

Capture these first:

1. Main screen
   - Show recording mode
   - Show resolution and bitrate
   - Show save path area

2. Countdown screen
   - Show the 5-second countdown
   - Show the microphone volume hint text

3. Recording control window
   - Show pause / resume
   - Show stop button
   - Show recording time

4. Support author window
   - Prefer the Chinese QR-code version for the first screenshot set

5. Fullscreen recording example
   - Show a completed recording result or playback frame

6. Window recording example
   - Show a single selected target window

## Optional screenshot set

Add these later if needed:
- Audio-only mode
- English UI
- Installer completion screen
- Portable folder contents

## Screenshot quality rules

- Keep the window centered and clean
- Avoid showing private desktop files
- Avoid showing personal chats, emails, or browser tabs
- Use consistent window size where possible
- Prefer crisp screenshots over compressed images
- Keep text readable at listing-page size

## Suggested publish order

For GitHub or itch.io:

1. Cover image
2. Main screen
3. Fullscreen recording example
4. Window recording example
5. Recording control window
6. Countdown screen
7. Support author window
