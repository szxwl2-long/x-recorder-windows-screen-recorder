# Microsoft Store Listing Draft

## Product Name

`X Recorder`

## Short Description

Open-source Windows screen recorder with fullscreen capture, window recording, microphone input, system audio, and audio-only mode.

## Full Description

`X Recorder` is a lightweight screen recorder for `Windows 10/11 x64`.

It is designed for users who want a simple desktop recording tool without a complicated workflow. The app supports fullscreen capture, single-window capture, microphone recording, system audio recording, and audio-only recording.

Main features:
- Fullscreen recording
- Window recording
- Audio-only recording
- Microphone capture
- System audio capture
- Chinese / English interface
- 2K / 1080P / 720P / 480P presets
- Recommended bitrate autofill
- Save folder selection
- 5-second countdown before recording starts
- Compact floating control panel during recording

Privacy summary:
- No online account required
- No automatic cloud upload
- No built-in analytics service
- Recordings are saved locally on the device

This project is open source under `GPL-3.0`. If the app is useful to you, you can support the author voluntarily inside the app.
