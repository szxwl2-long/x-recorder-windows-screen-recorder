## Short project description

`X Recorder` is an open-source Windows 10/11 screen recorder for fullscreen recording, window recording, audio-only recording, microphone input, system audio capture, and a Chinese / English user interface.

## What the software does

The project provides a practical Windows desktop recording tool for normal users who want to:
- record the full screen
- record a single target window
- record microphone audio
- record system audio
- export audio-only recordings

It is distributed as:
- a Windows installer
- a portable Windows build

## Intended users

The intended users are normal Windows desktop users who need a simple recorder without requiring a development environment.

## Open-source status

The project is publicly available on GitHub and licensed under `GPL-3.0`.

Source code:
- public repository
- public release downloads
- public license file

## Why signing is needed

The project is distributed directly to end users as a Windows desktop application. Code signing is needed to improve trust, reduce SmartScreen friction, and make public downloads easier and safer for users.

## Funding model

The software is free to use. The author does not rely on paid licensing. Users may support the project voluntarily through the in-app support page.
