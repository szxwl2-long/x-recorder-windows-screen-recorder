# Release Text

## GitHub Release Title

`X Recorder 1.0.0`

## Short Description

`X Recorder` is a Windows 10/11 x64 screen recorder with fullscreen recording, window recording, audio-only mode, microphone input, system audio capture, and Chinese / English UI.

## Release Notes

### English

Highlights:
- Fullscreen recording
- Window recording
- Audio-only recording
- Microphone and system audio options
- Chinese and English interface
- Installer build and portable build

Important notes:
- Target system: Windows 10/11 64-bit
- The installer is currently unsigned, so Windows may show a SmartScreen warning
- The portable version must remain as a full folder and should not be split

Files:
- `X-Recorder-Setup.exe`: recommended for most users
- `X-Recorder-Portable.zip`: portable version

## itch.io Page Copy

### Headline

`A simple Windows recorder for fullscreen, windows, microphone, and audio-only capture.`

### Body

`X Recorder` is a Windows desktop recorder built for Windows 10/11 x64.

It supports fullscreen recording, window recording, audio-only recording, microphone input, system audio capture, save-folder selection, bitrate presets, and a Chinese / English interface.

Recommended download:
- `X-Recorder-Setup.exe`

Portable users:
- Download `X-Recorder-Portable.zip`
- Extract the full folder before running

Important:
- Windows 10/11 64-bit only
- SmartScreen warning may appear because the installer is not yet code-signed
