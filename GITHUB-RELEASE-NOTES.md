## X Recorder 1.0.0

`X Recorder` is a Windows 10/11 x64 screen recorder for fullscreen capture, window recording, microphone input, system audio capture, and audio-only recording.

### Highlights

- Fullscreen recording
- Window recording
- Audio-only recording
- Microphone recording
- System audio recording
- Chinese / English interface
- Installer build and portable build

### Downloads

- `X-Recorder-Setup.exe`
  Recommended for most users.

- `X-Recorder-Portable.zip`
  Portable version for users who prefer not to install.

### Important Notes

- Target system: `Windows 10 / Windows 11 64-bit`
- The installer is currently **not code-signed**
- Windows may show a `SmartScreen` warning on first launch
- The portable version must be extracted as a full folder
- Do **not** copy only `WindosRecorder.exe`
