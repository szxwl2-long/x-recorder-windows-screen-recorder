## Title

`X Recorder - Windows Screen Recorder`

## Short description

`A Windows 10/11 screen recorder with fullscreen capture, window recording, microphone input, system audio, and audio-only mode.`

## Full description

`X Recorder` is an open-source Windows screen recorder built for `Windows 10/11 x64`.

It supports fullscreen recording, window recording, audio-only recording, microphone input, system audio capture, bitrate presets, save-folder selection, and a Chinese / English interface.

## Features

- Fullscreen recording
- Window recording
- Audio-only recording
- Microphone recording
- System audio recording
- Chinese / English UI
- 2K / 1080P / 720P / 480P presets
- Installer build and portable build

## Downloads

- `X-Recorder-Setup.exe`
  Recommended for most users

- `X-Recorder-Portable.zip`
  Portable version for users who prefer not to install

## Important notes

- Windows 10 / Windows 11 64-bit only
- Current public installer is unsigned
- Windows may show a SmartScreen warning on first launch
- Portable version must be extracted as a full folder
- Do not copy only the EXE file
