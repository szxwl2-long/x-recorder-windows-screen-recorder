# Project Intro

## English

`X Recorder` is an open-source Windows screen recorder for `Windows 10/11 64-bit`.

It supports fullscreen recording, window recording, audio-only recording, microphone input, system audio capture, and a Chinese / English interface.

The goal of this project is to provide a practical and easy-to-use desktop recording tool that normal users can download and run without needing a development environment.

The software is free to use. If users find it helpful, they can support the author voluntarily through the in-app support page.

## 中文

`X Recorder` 是一款面向 `Windows 10 / 11 64 位` 的开源录屏软件。

它支持全屏录制、窗口录制、单独录音、麦克风录制、系统声音录制，以及中英文界面。

这个项目的目标，是提供一个普通用户也能直接下载使用、无需开发环境即可运行的实用桌面录制工具。

软件可免费下载和使用。如果用户觉得好用，可以通过软件内的支持页面自愿打赏作者。
