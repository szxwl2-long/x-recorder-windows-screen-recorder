## 一、项目目标

用户最初希望制作一个 Windows 应用程序，用于高清录像，主要功能包括：

- 全屏录像。
- 指定窗口录像。
- 录制过程中可以同时录入麦克风声音。
- 可以单独录制音频。
- 做成普通用户能使用的软件，不依赖开发环境。
- 支持中文和英文。
- 可以打包成可安装版本，也可以保留便携版。
- 软件可发布给别人使用，并允许用户自愿打赏支持作者 long-X。

## 六、已实现功能记录

录制功能：

- 支持全屏录制。
- 支持窗口录制。
- 支持单独录音。
- 支持麦克风录入开关。
- 支持系统声音录入开关。
- 支持麦克风音量调节。
- 支持系统音量调节。
- 麦克风音量支持放大倍率按钮，循环倍率为 100、200、300、400、500，再回到 100。
- 支持暂停、继续、停止。
- 开始录制后有 5 秒倒计时。
- 录制开始后主界面缩小/隐藏，使用小控制窗口操作。

分辨率和码率：

- 支持 2K、1080P、720P、480P。
- 分辨率对应推荐码率：

```text
2K     -> 12000 kbps
1080P  -> 6000 kbps
720P   -> 3000 kbps
480P   -> 1500 kbps
```

- 选择分辨率时自动填入推荐码率。
- 用户可手动调整码率。
- 容量预估会根据码率、帧率和保存分区容量进行计算。
- 容量预估放在分辨率区域下方。
- 可自行调整码率的提醒放在码率区域上方。

保存和文件名：

- 支持设置保存路径。
- 支持打开视频文件夹按钮。
- 删除了之前“下一个文件”的输入框，改为“记录名称”。
- 文件名格式已改为 `X-` 前缀。
- 年份格式去掉前面的 `20`，例如 2026 年显示为 `26`。
- 用户输入的记录名称会保存，下次打开软件自动显示上一次输入。
- 如果用户没有输入，则保持为空。
- 清空输入框后，下拉选项会显示最近三次改动的名称。

语言：

- 软件首次打开会弹出语言选择窗口。
- 有两个选项：

```text
中文
English
```

- 选择中文后，主界面、窗口选择、录制控制、提示文本等应显示中文。
- 选择英文后，主界面、窗口选择、录制控制、提示文本等应显示英文。
- 曾修复英文状态下作者标签仍显示中文的问题。
- 曾修复英文状态下主界面和浮窗里部分选项仍是中文的问题。

界面：

- 参考用户提供的录屏软件风格进行 UI 优化。
- 主界面采用卡片式设计。
- 保持主窗口内容尽量在一个界面显示，不依赖左侧上下滚动。
- 录制模式包括：

```text
全屏录制 / Fullscreen
窗口录制 / Window
单独录音 / Audio Only
```

- 窗口录制时才显示目标窗口选择行。
- 用户希望窗口录制多出一行时界面向上压缩，而不是把下面内容向下挤出窗口。
- 主界面底部只保留开始录制按钮，删除了无用的停止录制按钮。
- 顶部右侧显示作者 long-X。
- 顶部中间加入支持作者/打赏入口。

麦克风设备显示：

- 用户希望麦克风选项不要显示复杂设备路径。
- 目标显示风格类似：

```text
麦克风 (Realtek High Definition Audio)
立体声混音 (Realtek High Definition Audio)
```

- 已做过设备名简化优化。

控制窗口：

- 曾有右下角透明录制时间浮层。
- 后来因可能被录进视频，用户要求删除浮窗。
- 当前方向：删除右下角浮窗，只使用小控制窗口。
- 小控制窗口使用暂停/继续/停止图标。
- 图标要求统一风格、不要红色、描边中空，和主界面统一。

窗口选择：

- 曾出现两个关闭按钮。
- 已修复为只保留系统窗口关闭按钮。

错误处理：

- 曾出现启动失败白屏、崩溃。
- 曾增加启动异常提示逻辑。
- 曾出现关闭窗口后无法设置可见性、无法调用 Show/ShowDialog、WindowInteropHelper.EnsureHandle 的报错。
- 修复方向是避免对已经关闭的 WPF Window 再调用显示/隐藏相关操作。

## 九、安装包和兼容性记录

用户曾询问软件如果制作成安装包，是否所有 Windows 电脑都能正常使用。

结论：

- 不建议承诺所有 Windows。
- 当前项目基于 .NET 8 和现代 Windows API，更适合 Windows 10/11 64 位。
- Windows 7 不适合继续支持，原因包括 .NET 8、录屏相关 API、ScreenRecorderLib 依赖、系统组件差异。
- Windows 11 若无法运行，常见原因可能是 VC++ Runtime、Media Foundation、显卡驱动、系统权限或安全策略。

当前安装包做法：

- 使用 Inno Setup。
- 安装包语言可选择中文和英文。
- 安装包包含 VC++ x64 Runtime：

```text
installer-assets\vc_redist.x64.exe
```

- 安装时如果系统缺少 VC++ x64 Runtime，会自动安装。

安装包输出：

```text
dist\installer\X-Recorder-Setup.exe
```

安装包脚本：

```text
WindosRecorder.iss
```
